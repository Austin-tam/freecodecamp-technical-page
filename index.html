Introduction

Python is an interpreted, object-oriented programming language. Created by Guido van Rossum and released in 1991, Python is one of the most popular choices for backend development, being employed by websites such as Instagram, Uber, Dropbox, and many more.

The main feature of python is that it is made to be beginner friendly and easy to learn. Other features of the language include:

  • Easy-to-read: The simplistic syntax makes it easy and quick to read the code
  • Easy-to-maintain: Since the code is easy to read, it is therefore easy to maintain
  • Portable: Python can run on many different hardware platforms and shares the same interface across all of these platforms
  • Large Standard Library: Python has a large standard library which provides a set of many useful built-in modules and functions for a variety of uses

Syntax

The definition of syntax is "the arrangement of words and phrases to create well-formed sentences in a language". We can apply this definition to programming languages as well. Syntax, in the context of a programming language, refers to the structure of the language, and the arrangement of words and phrases. Although Python was originally developed as a teaching language, its easy-to-use nature and clean syntax has appealed to beginners and experts as well. Python uses English words in places where other languages would use punctuation.

For example, take the following Java code:

int[] values = new int[100];
for (int i=0; i<values.length; i++) { }

Now compare it to the following Python code:

values = [0] * 10
for i in range(0, len(values), 1):

As you can tell, the for loop declaration in Python is "easier" to read since it uses more English words and one can instinctively understand what is happening. Whereas with the Java code, one would have to look through each section of the for loop to understand each part.

Another example is Python's use of Not in place of ! that many other languages use. With a larger code base, finding and reading the keyword Not is easier than ! which makes it harder to miss and harder to make errors.

Variables

In programming, variables are used to store data values. Python is a strongly and dynamically typed language. Strong meaning that the type of a value does not change in unexpected ways, and dynamic meaning that Python checks the types of values at runtime rather than at compile time. Because of this, variables in Python code are very easy to create and initialize.


For example, a simple variable declaration can look like this:

x = 0

This tells Python to create an integer object with a value of 0 and have the variable "x" point to that object. Similarly, when you declare strings like this:

y = "text"

Python creates a string object with the value of "text" and has the variable "y" point to that object.

Python has very little restrictions in terms of variable names. They can be any length and can consist of any combination of uppercase and lowercase letters (A-Z, a-z), digits (0-9), and the underscore character (_). The main restriction is that the first character of a variable name cannot be a digit. The only other restriction is that variable names in Python cannot be the same as any of the keywords that Python has reserved for special functions. These keywords include:

  • False
  • True
  • if
  • None
  • not
  • and
  • def
  • for
  • break
  • while
  • try
